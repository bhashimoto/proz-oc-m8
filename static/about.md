# Ordens de Compra — M8
versão 0.1.0

Aplicativo para automação de criação de ordens de compra no sistema ERP M8.

## Instruções de Uso

1. **Importar OC** — selecione a planilha modelo preenchida.
2. Revise as ordens na tabela.
3. **Criar OCs no M8** — envia as ordens para o sistema. Credenciais serão solicitadas caso ainda não tenham sido informadas.

## Planilha modelo

Acesse **Arquivo → Salvar planilha modelo** para baixar o arquivo Excel padrão.
